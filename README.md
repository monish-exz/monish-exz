##  Selected Projects

### И ML-Dataset-Lint
A lightweight, rule-based tool to catch dataset issues
before training machine learning models.

It checks for:
- missing values, duplicates, and constant columns
- class imbalance and rare labels
- potential data leakage signals
- basic bias and trainability risks

The goal is to keep it simple, explainable,
and useful for learners and early ML projects.

**Tech:** Python 路 Machine Learning 路 Data Quality
